# IRIS integrate Android library 

The IRIS integrate Android library predicts if a person may be impaired due to substance use.

## Documentation

Documentation is [available here](https://sighticanalytics.github.io/iris-integrate-android/).

## Adding IRIS integrate to your project

Add IRIS integrate as a dependency to your `build.gradle` file.

```kotlin
implementation("com.sightic:irisintegrate:1.0.0")
```

The latest version can be found on [Maven Central](https://central.sonatype.com/artifact/com.sightic/irisintegrate/overview).

## Quickstart App

There is a [Quickstart App](https://github.com/SighticAnalytics/iris-integrate-quickstart-app-android) sample project available that shows how to integrate the SDK in an Android app.
